Drug Inventory System:
To run  the project:
1. Create a Database named Drugs in the PostgreSQL database 

Tehcnology Stack: Spring Boot, Spring MVC, Hibernate, PostgreSQL

1.The project creates different drugs and mechanisms used to create the drugs and posts to the database.
2.It allows the user to search for a specific Drug or Mechanism based on the search Query
3.It establishes a Many to Many realtionship between Drugs and its corresponsing Mechanisms. That is one Drug can be produced by Many Mechaims and
One mechanism can be used to Produce many Drugs
